# House Tour — Interactive 3D Viewer

A browser-based interactive 3D viewer for a 2BHK apartment floor plan built in Blender.

## Features
- Auto-rotating 3D model
- Touch controls (drag, pinch, pan) — works on any phone
- 3 camera presets (Top, Front, Isometric)
- Room dimensions panel
- No app or account needed
